# Aşama 19.5 — Global Mobil Randevu Deneyimi

Bu sürüm 19.4 çalışan proje temel alınarak hazırlanmıştır.

Eklenenler:
- Tüm sayfalarda mobil alt randevu barı
- Ara, WhatsApp ve Randevu Al seçenekleri
- WhatsApp için hazır mesajlar
- GA4 dönüşüm takibine uygun data kaynakları
- iPhone güvenli alan desteği
- Mobilde sağ dock ve eski alt bar çakışmasının kaldırılması
- Çerez panelinin mobil bardan yukarı taşınması

Kontroller:
- `npm run typecheck` başarılı
- `npm run lint` başarılı
- `npm run build` kod derleme aşamasında ortam süresi sınırına takıldı

Kurulum:
```bash
npm install
npm run dev
```
