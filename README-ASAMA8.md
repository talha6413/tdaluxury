# Aşama 8 — Premium Galeri, Kampanyalar ve SSS

Bu aşamada üç zayıf iç sayfa yayın kalitesine çıkarıldı:

- `/galeri`: fotoğraflı masonry benzeri galeri, seçili öncesi/sonrası alanı ve güçlü CTA
- `/kampanyalar`: görselli kampanya kartları, kampanya kullanım adımları ve WhatsApp dönüşüm alanı
- `/sss`: kategori bazlı premium akordeonlar, FAQPage JSON-LD ve danışmanlık CTA'sı

## Test

```bash
npm install
npm run typecheck
npm run lint
npm run build
npm run dev
```

Kontrol adresleri:

- http://localhost:3000/galeri
- http://localhost:3000/kampanyalar
- http://localhost:3000/sss
