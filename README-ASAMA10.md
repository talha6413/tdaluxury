# TDA Luxury — Aşama 10 Premium Motion

Bu sürümde:
- Scroll ile küçülen ve cam efekti kazanan navbar
- Sayfa üstü altın scroll progress çizgisi
- Hero giriş animasyonları ve hafif parallax
- Hizmet kartlarında zoom, yükselme, altın çerçeve ve CTA geçişleri
- Parmak ve fareyle sürüklenebilen önce/sonra slider'ları
- Görünür olduğunda çalışan animasyonlu sayaçlar
- Bölüm bazlı scroll reveal
- `prefers-reduced-motion` erişilebilirlik desteği

Kontroller:
- npm run typecheck: başarılı
- npm run lint: başarılı
- npm run build: başarılı
- 55 rota üretildi

Kurulum:
```bash
npm install
npm run dev
```
