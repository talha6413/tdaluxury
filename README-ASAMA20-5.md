# Aşama 20.5 — Premium Hizmet Detay Sistemi

Bu sürüm 20.4 temel alınarak hazırlanmıştır.

Eklenenler:
- Tüm standart hizmet detay sayfalarında kategoriye göre değişen hızlı bilgi bandı
- Lazer, cilt bakımı, kalıcı makyaj ve diğer hizmetler için farklı süreç vurguları
- Sticky bilgi kartında premium altın detaylar
- Açık SSS kartlarında güçlendirilmiş görünüm
- İlgili hizmet kartlarında hover/yükselme efekti
- Mobil ve tablet uyumlu hızlı bilgi düzeni
- Global iletişim paneli bulunduğu için hizmet sayfalarındaki eski tekil WhatsApp balonu kaldırıldı

Kontrol edilen komutlar:
- npm run typecheck: başarılı
- npm run lint: başarılı
- npm run build: kod derleme başladı ancak çalışma ortamı süre sınırında tamamlanmadı
