# Aşama 20.4 — Global Akıllı Güzellik Danışmanı

Bu sürümde:

- Akıllı danışman ana sayfadan global layout seviyesine taşındı.
- Tüm sayfalarda çalışır.
- Kadın / erkek seçimi, hizmet seçimi ve ihtiyaç seçimi içerir.
- Sonuçlar hazır WhatsApp mesajına dönüştürülür.
- Mobilde alt randevu barıyla çakışmayacak şekilde konumlandırılır.
- WhatsApp dönüşüm ölçümü mevcut ConversionTracker ile uyumludur.

Kurulum:

```bash
npm install
npm run dev
```
