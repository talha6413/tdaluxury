# TDA Luxury — Aşama 6

Bu sürüm Aşama 5 üzerine hazırlanmıştır.

## Yenilikler
- Premium Hakkımızda sayfası
- Fotoğraflı ve kategori etiketli premium Blog sayfası
- 6 özgün blog içeriği
- Dinamik blog metadata ve statik parametreler
- Premium blog detay tasarımı
- İlgili yazılar ve WhatsApp CTA
- Blog URL'lerinin sitemap'e eklenmesi
- Mobil uyumlu yeni sayfa stilleri

## Çalıştırma
```bash
npm install
npm run dev
```

## Kontrol
```bash
npm run typecheck
npm run lint
npm run build
```

`typecheck` ve `lint` başarıyla geçmiştir. Üretim derlemesi kod derleme ve TypeScript aşamalarını başarıyla geçti; bu çalışma ortamında statik sayfa üretimi süre sınırına takıldı.
