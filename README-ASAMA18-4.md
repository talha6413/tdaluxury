# Aşama 18.4 — Blog SEO Sistemi

Bu sürüm, Aşama 18.3 tam proje paketi temel alınarak hazırlandı.

## Eklenenler

- Blog yazısı sayısı 6'dan 10'a çıkarıldı.
- Her yazıya yayın ve güncelleme tarihi eklendi.
- Her yazıya özel anahtar kelime alanları eklendi.
- Her yazıya ilgili hizmet bağlantıları eklendi.
- Article ve BreadcrumbList JSON-LD eklendi.
- İlgili yazılar aynı kategori öncelikli hale getirildi.
- Blog detay sayfasında hizmetlere iç bağlantı alanı eklendi.
- Sitemap yeni blog yazılarını otomatik içeriyor.

## Kontroller

- npm run typecheck: başarılı
- npm run lint: başarılı
- npm run build: başarılı
- 59 statik sayfa üretildi.
