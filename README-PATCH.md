# TDA Luxury Premium Cilt Bakımı SEO Patch

Bu paket yalnızca şunları değiştirir:

- app/cilt-bakimi/page.tsx
- components/cilt-bakimi/SkinCareFaq.tsx
- public/images/cilt-bakimi/*

Ana sayfaya, globals.css dosyasına, Navbar'a ve diğer sayfalara dokunmaz.

## Kurulum

ZIP içindeki `app`, `components` ve `public` klasörlerini mevcut `tdaluxury` klasörünüzün içine sürükleyin ve dosya değişimini onaylayın.

Sonra:

```bash
npm run dev
```

Kontrol:

http://localhost:3000/cilt-bakimi

Geçici SVG görselleri profesyonel çekimlerden sonra aynı dosya adları korunarak gerçek görsellerle değiştirebilirsiniz.
