# Aşama 19.1 — Global Hızlı İletişim Dock

Bu sürümde sağdaki Ara / WhatsApp / Instagram / Yol Tarifi / Yukarı Çık paneli tüm sayfalarda global olarak gösterilir.

- Root layout içine taşındı.
- Ana sayfa ve hizmet sayfalarındaki tekrarlar kaldırıldı.
- Masaüstü ve tablet genişliklerinde sağda sabit kalır.
- Mobilde mevcut mobil iletişim deneyimini engellememek için gizlenir.
- Yol Tarifi doğrudan Google Maps aramasını açar.
