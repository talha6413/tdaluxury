# Aşama 15.4 — Mobil, Video, Erişilebilirlik ve Final Sağlamlaştırma

Bu sürümde:

- Hero tanıtım videosu gerçek modal içinde çalışır.
- Modal ESC, kapatma düğmesi ve dış alana tıklamayla kapanır.
- Video açıldığında sayfanın arka plan kaydırması kilitlenir.
- Masaüstü alt iletişim barı aşağı kaydırmada gizlenir, yukarı kaydırmada görünür.
- Mobil menü açıldığında sayfa kaydırması kilitlenir ve ESC ile kapanır.
- Mobil menü ve modal için erişilebilirlik nitelikleri güçlendirildi.
- Küçük ekranlarda Beauty Assistant yüksekliği ve taşma davranışı düzeltildi.
- Hareket azaltma tercihi desteklendi.

## Çalıştırma

```bash
npm install
npm run dev
```

npm run typecheck
npm run lint
